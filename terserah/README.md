# Kelas X13 - Template Website

Template sederhana untuk halaman kelas X13 (HTML, CSS). Dirancang agar bekerja tanpa JavaScript dan ramah untuk tampilan Android.

Files:
- index.html — halaman utama (non-JS friendly)
- styles.css — styling responsif dan touch-friendly
- README.md — instruksi singkat

Cara pakai:
1. Buka [index.html](index.html) di browser (klik dua kali atau gunakan Live Server).
2. Untuk mengubah pengumuman, edit element `<ul id="announcementList">` di [index.html](index.html).

Catatan penting:
- Situs ini dibuat agar fungsional tanpa JavaScript menggunakan teknik CSS-only untuk navigasi pada perangkat mobile.
- Tampilan dioptimalkan untuk Android (tap target lebih besar, touch-action, dan responsif).

Modifikasi:
- Ubah warna atau tata letak di [styles.css](styles.css).
